# Vehicle Vault Mobile App

Vehicle Vault is a mobile application available for IOS and Android that allows users to track information about their vehicles, making it easier than ever to keep up with vehicle maintenance. This is a collaborative project created by Brianna Geiger, Alyssa Rupard, Benjamin Foreman, Lucas Czarnecki, Nolan Ethridge, and Luke Orow.

This project was created with:
- React Native
- Node.js
- JS
- Expo
- Firebase
- Android Studio
- Figma
- AWS
- SmartCar API
- NHTSA API
